Prerequisites

Note: Known-good versions of the prequisites are available is the git repository for this workshop.

All

  • Download the Paho MQTT client from http://www.eclipse.org/paho/clients/tool/
  • Download or clone a copy of the workshop repository to your home directory, from https://github.com/InfernoEmbedded/esp-workshop
  • If you have a firewall enabled, allow traffic on port 1883 (MQTT)
  • Install NodeJS 4 (not 3 or 5), saved in esp-workshop/windows_x64 - https://nodejs.org/en/download/package-manager/
  • Install NodeRed from http://nodered.org/docs/getting-started/installation.html
  • Flash the board with NodeMCU firmware (only required is your board does not already have the NodeMCU firmware installed)
    • Download and extract the latest ESP8266 NONOS SDK from https://espressif.com/en/support/download/sdks-demos
    • Build the firmware using the Cloud build service https://nodemcu-build.com, Include:
      • ADC
      • bit
      • DHT
      • enduser_setup
      • file
      • GPIO
      • HTTP
      • MQTT
      • net
      • node
      • 1Wire
      • PWM
      • timer
      • UART
      • WiFi
      • Flash the firmware, this example uses the firmware included in the repository, but you may want your own (see above) instead: esptool.py --port /dev/ttyUSB0 write_flash -fm dio -fs 32m 0x00000 ~/esp-workshop/nodemcu-master-15-modules-2016-07-22-03-05-09-float.bin 0x7c000 ~/esp-workshop/sdk/ESP8266_NONOS_SDK/bin/esp_init_data_default.bin

Windows

  • Install CH341 Windows Drivers, saved in esp-workshop/windows_x64/drivers - http://www.wch.cn/download/CH341SER_ZIP.html
  • Install ESPTool, already installed in esp-workshop/windows_x64, this is only needed if you need to flash the ESP8266 module. https://github.com/igrr/esptool-ck/releases
  • Install the Java Runtime Environment (JRE) - http://www.oracle.com/technetwork/java/javase/downloads/index.html

Linux

  • Install Mosquitto (Fedora: dnf install mosquitto, Debian/Ubuntu: apt-get install mosquitto)
  • Install ESPTool, this is only needed if you need to flash the ESP8266 module. (Fedora: dnf install esptool, Debian/Ubuntu: apt-get install esptool)
  • Install java (Fedora: dnf install java, Debian/Ubuntu: apt-get install default-jre)
  • Add your user the the dialout group so you can access the serial port (edit /etc/group, sudo gpasswd -a ${USER} dialout && newgrp dialout)

Kit Contents

Please check to ensure you have all the parts to your kit. Clockwise from the top, the parts are:

  • Breadboard Jumper Leads
  • MicroUSB Cable
  • ESP8266 Module (Wemos D1 Mini or clone)
  • Breadboard
  • 37 Module Sensor Kit
